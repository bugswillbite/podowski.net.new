Decorate My Room

Decorate My Room is a browser-based room decoration game built with HTML, CSS, and JavaScript. Players design their own personal space by selecting furniture, decorations, and accessories, experimenting with different combinations to create a room that reflects their own style.

Role

Game Designer, UI Designer, Illustrator & Frontend Developer

Tools

HTML, CSS, JavaScript, Photoshop

Play on itch.io

Decorate My Room is an interactive browser game that transforms interior decorating into a playful creative experience. Rather than completing objectives or solving puzzles, players freely experiment with furniture, decorations, and room layouts to create spaces that feel uniquely their own.


Built entirely with HTML, CSS, and JavaScript, the project explores how web technologies can support engaging, interactive experiences through responsive interfaces, layered artwork, and dynamic customization systems. Every design decision was centered around encouraging creativity, experimentation, and personal expression.

The core gameplay revolves around customization. Players can browse multiple furniture, decoration, and accessory options, mixing and matching objects to build a room that reflects their own aesthetic preferences.



Every selectable object updates the environment instantly, allowing players to experiment with countless combinations without interrupting the creative process. The experience is designed to feel playful, intuitive, and open-ended rather than goal-oriented.

The interface was designed around simplicity and immediate feedback. JavaScript dynamically swaps furniture and room elements in real time, allowing players to instantly preview different layouts without navigating complicated menus.



Careful attention was given to responsiveness, layering, and interaction design to ensure every customization feels seamless. Smooth transitions and organized controls help maintain a relaxed creative workflow while encouraging players to continue experimenting.

Every room asset was illustrated to create a cohesive visual style inspired by cozy interiors and personal living spaces. Furniture, decorations, plants, lighting, and decorative objects were designed to work together regardless of which combinations players choose.



Typography, color palettes, iconography, and interface elements were intentionally understated so the customizable room remains the focal point. The result is an inviting interface that supports creativity without overwhelming the player.

Decorate My Room gave me the opportunity to combine frontend development, illustration, interaction design, and game design into a single browser-based experience. Building the customization system required balancing clean interface design with responsive JavaScript interactions so decorating remained fluid and enjoyable.


Developing the project strengthened my experience with HTML, CSS, and JavaScript while demonstrating how browser technologies can be used to create engaging interactive applications beyond traditional websites. The project also reinforced my interest in designing systems that encourage creativity, experimentation, and player expression.
